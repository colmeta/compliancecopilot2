## üèóÔ∏è Technical Architecture

### **Core Technologies**
- **Backend**: Flask + Celery for async processing
- **AI Engine**: Google Gemini (Flash/Pro/Ultra) with intelligent routing
- **Vector Store**: ChromaDB with sentence-transformers
- **Database**: PostgreSQL with SQLAlchemy
- **Cache**: Redis for response caching
- **OCR**: Tesseract, Google Vision, AWS Textract
- **Real-time**: Flask-SocketIO for collaboration

### **Key Architectural Decisions**
1. **Multi-Tenant Isolation**: Each user gets their own ChromaDB collection
2. **Async Processing**: All heavy operations run on Celery workers
3. **Intelligent Routing**: Tasks automatically routed to optimal AI model
4. **Semantic Caching**: Responses cached with similarity matching
5. **Modular Design**: Easy to add new accelerators and features

## üöÄ Getting Started

### **For Users**
```bash
# 1. Clone repository
git clone <repo-url>

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set up environment
cp .env.example .env
# Edit .env with your API keys

# 4. Initialize database
flask db upgrade

# 5. Start services
# Terminal 1: Flask server
python run.py

# Terminal 2: Celery worker
celery -A celery_worker.celery_app worker --loglevel=info

# Terminal 3: ChromaDB
chroma run --path ./chroma_data
```

### **For Developers**
```python
from app.data_entry import KeystoneEngine
from app.planning import PlanningEngine
from app.writing import HumanTouchWriter
from app.ai_optimization import get_model_router

# Use Data Keystone Engine
engine = KeystoneEngine(confidence_threshold=0.85)
result = engine.process_document(document, schema, output_config)

# Use Planning Engine
planner = PlanningEngine()
plan = planner.create_plan(task_description, task_type, context)
planner.approve_plan(plan)

# Use Human Touch Writer
writer = HumanTouchWriter()
voice_profile = writer.analyze_voice(samples, org_name, industry)
humanized = writer.humanize_content(content, writing_style)

# Use Model Router
router = get_model_router()
selection = router.route_task(
    task_type='analysis',
    task_complexity='complex',
    user_tier='enterprise',
    context_size=50000
)
```
